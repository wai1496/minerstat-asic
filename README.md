# minerstat for ASIC

**minerstat for ASIC** is mining monitoring and management software for your **Antminer** and/or **Baikal** machines.

<img src="https://cdn.rawgit.com/coinscrow/minerstat-asic/master/docs/minerstat_ui.png" width="90%">
